# NayePankh Foundation Student Management System

## Project Overview

This is a Python-based Student Management System developed for the NayePankh Foundation internship task.

The project helps manage student records and provides features such as adding, viewing, searching, filtering, and deleting student information.

## Features

* Add Student Records
* View All Students
* Search Student by Name
* Filter Students by Course
* Delete Student Records
* Generate Reports
* Store Data Permanently Using JSON Files

## Technologies Used

* Python
* JSON File Handling
* VS Code
* Git & GitHub

## Project Structure

NayePankh_Student_Management

* main.py
* students.json
* report.txt
* README.md

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:   python main.py
4. Select options from the menu.

## Sample Menu

1. Add Student
2. View Students
3. Search Student
4. Filter By Course
5. Delete Student
6. Generate Report
7. Exit
